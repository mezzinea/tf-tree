# tf-tree

**tf-tree** is a lightweight, browser-based viewer for Terraform dependency graphs.  
It allows you to load Terraform-generated `.dot` files and visually explore resource relationships, dependencies, and execution flow.

## Features

- Load one or multiple `.dot` files
- Interactive graph navigation
- Click any resource to highlight:
  - **Child nodes** (resources depending on the selected resource)
  - **Parent nodes** (resources the selected resource depends on)
- Automatic dependency counters
- Resource search with auto-focus
- Clickable dependency lists
- Zoom in / out / reset controls
- Faded unrelated nodes for clarity
- Rounded, styled nodes with visual highlighting
- Graph direction normalized to **Left â†’ Right (LR)**
- Runs fully in the browser (no backend)

---

## Generate a Terraform Graph (.dot)

From your Terraform project directory:

```bash
terraform init
terraform graph > graph.dot
```

This generates a Graphviz DOT file containing resource dependencies.

## How to Use

1. Open the UI  
   ðŸ‘‰ https://mezzinea.github.io/tf-tree/

2. Upload one or more `.dot` files.

3. Select a graph from the dropdown.

4. Click any node to explore its dependencies.

5. Use search and zoom tools for navigation.

6. Use **Clear Highlight** to reset the view.
